Hardware Components

IoT devices and sensors used in this project

Arduino UNO R3

Main microcontroller board with ATmega328P processor for data processing and communication.

MQ135 Gas Sensor

Detects NH3, NOx, alcohol, benzene, smoke, CO2 with high sensitivity and fast response.

DHT11 Sensor

Digital temperature and humidity sensor with calibrated digital signal output.

LDR Sensor

Light Dependent Resistor for measuring ambient light intensity and brightness levels.
